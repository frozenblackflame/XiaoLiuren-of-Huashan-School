# 华山派小六壬

华山派小六壬

## 功能特性

- 支持小六壬占卜方式，并使用华山派小六壬216占文来进行解卦

## 环境依赖

- 请点击  [此处](https://dotnet.microsoft.com/download/dotnet/thank-you/runtime-desktop-8.0.1-windows-x64-installer) 下载并安装 .NET 8 运行库（.NET 桌面运行时 8.0.1 x64）。

## 使用说明

1. 选择您想要查询的月日时后点击确认。
2. 点击随机按钮随机月日时后自己给出占文。
3. 点击重置按钮开始新一轮占卜。
